CREADOR DE PAC PARA SECRETARIOS


Políticas de Uso de Información

Bajo ningún punto de vista esta web almacena o distribuye información sensible.

Información de las Tecnologías Utilizadas

  • Base de datos: MongoDB Atlas
  • Servidor: Render
  • Backend: Node.js con Express
  • Frontend: HTML, JavaScript, CSS
  • Frontend alojado en: GitHub Pages

Permisos

*Los permisos suministrados por el usuario permiten a esta web:

  • Leer los correos electrónicos (no modificarlos)
  • Leer el nombre de usuario y dirección de correo electrónico (utilizados para mostrar la foto de perfil, nombre y correo al inicio de la web)

Lógica Programática

*Esta web:

  • Busca los datos para armar los PAC en los correos del usuario logueado
  • Una vez que se arma el PAC, no se guarda información sensible de ningún agente

¿Qué se almacena en la base de datos?
Se almacena del usuario logueado: nombre y correo electrónico para reconocerlo al reingresar, evitando tener que loguearse constantemente.

Importante

No se guardan en el servidor los PAC generados por esta web. Son de creación momentánea en la computadora del usuario.
